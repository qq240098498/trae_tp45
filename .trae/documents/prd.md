## 1. 产品概述

供应链优化系统是一套面向制造企业的全链路智能决策平台，覆盖采购、生产、交付三大核心环节。通过实时监控链路运行状态、分析库存健康度、识别瓶颈与风险、结合历史与季节性数据给出采购建议、协调供应商与生产排期，帮助决策者在成本、时效、风险三者之间取得最优平衡。

- 目标用户：供应链经理、采购专员、生产计划员、仓储管理员
- 核心价值：将分散的供应链数据整合为可操作的洞察，降低库存成本15%-25%，减少延误风险30%以上

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 供应链经理 | 查看全局仪表盘、审批采购建议、调整风险阈值 |
| 采购专员 | 查看采购建议、管理供应商信息、发起采购订单 |
| 生产计划员 | 查看生产排期、协调供应商交付时间、调整排产计划 |
| 仓储管理员 | 查看库存分析、管理安全库存、上报异常 |

### 2.2 功能模块

1. **全局监控仪表盘**：供应链全链路可视化、关键KPI卡片、实时状态流、异常告警
2. **库存分析中心**：周转率分析、安全库存水位图、ABC分类矩阵、库存预警
3. **风险与瓶颈识别**：链路瓶颈定位、延误风险评估、风险热力图、预警推送
4. **智能采购建议**：历史趋势分析、季节性因子计算、采购量推荐、成本模拟
5. **排程协调中心**：供应商交付时间线、生产排期甘特图、冲突检测、协调建议
6. **三维度平衡分析**：成本-时效-风险雷达图、方案对比、决策建议

## 3. 核心流程

用户打开系统后，首先在全局监控仪表盘了解链路整体健康状态。若发现异常告警，可进入风险与瓶颈识别页面定位问题。日常工作中，采购专员查看智能采购建议完成补货决策，生产计划员在排程协调中心对齐供应商交付与生产排期。所有决策均可通过三维度平衡分析进行成本-时效-风险的权衡验证。

```mermaid
    "日常运营" --> "库存分析中心"
    "库存分析中心" --> "智能采购建议"
    "智能采购建议" --> "三维度平衡分析"
    "三维度平衡分析" --> "采购决策"

    "排程协调中心" --> "三维度平衡分析"
    "三维度平衡分析" --> "排程调整决策"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色工业风（#0F1419 底色），搭配琥珀色(#F59E0B)作为主要强调色、青色(#06B6D4)作为辅助信息色、红色(#EF4444)作为风险警示色
- **按钮风格**：圆角矩形(8px)，悬停时有微妙的光晕效果
- **字体**：标题使用 JetBrains Mono（工业/数据感），正文使用 Noto Sans SC（中文优化）
- **布局风格**：左侧固定导航栏 + 右侧内容区，内容区采用卡片网格布局
- **图标风格**：线性图标（Lucide），统一2px描边，配合数据可视化使用填充色块

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 全局监控仪表盘 | 链路状态流 | 水平流向图，节点圆形+连接线，绿色/黄色/红色状态灯 |
| 全局监控仪表盘 | KPI卡片组 | 4列网格卡片，每卡含图标+数值+趋势箭头+迷你折线图 |
| 全局监控仪表盘 | 异常告警流 | 右侧面板，时间轴式列表，图标区分级别 |
| 库存分析中心 | 周转率趋势图 | 面积图，渐变填充，虚线标注季节区间 |
| 库存分析中心 | 安全库存水位图 | 水平柱状图，三色带标识危险/预警/安全区间 |
| 库存分析中心 | ABC分类矩阵 | 散点气泡图，气泡大小表示库存量 |
| 风险与瓶颈识别 | 风险热力图 | 网格热力图，红-黄-绿渐变色 |
| 智能采购建议 | 季节性因子图 | 柱状图+折线叠加，阴影标注旺季 |

| 排程协调中心 | 甘特图 | 横向条形图，可拖拽，冲突区域标红 |
| 三维度平衡分析 | 雷达图 | 三角形雷达，填充半透明色，多方案叠层对比 |

### 4.3 响应式设计

- 桌面优先设计，最低支持1280px宽度
- 1920px以上充分利用空间展示更多数据列
- 移动端折叠侧边栏为底部Tab导航，卡片改为单列布局

### 4.4 动效设计

- 页面加载：卡片依次淡入（stagger animation，间隔80ms）
- 数据更新：数值变化时使用数字滚动动效
- 状态变更：节点颜色变化使用平滑过渡（300ms ease）
- 悬停反馈：卡片轻微上浮+阴影加深
- 异常告警：高风险项目脉冲呼吸灯效果
